Speedbock Technologies Docs
===========================

**Speedbock** (/spiːd-bɒk/) docs is the official page for the companies' devices.
The documentation gathers details on the hardware and the software for each device designed by Speedbock Technologies.

Check out the :doc:`hardware` for a guide on how to use our devices. Go to :doc:`software` section for information regarding
our Python module. We also have a :doc:`api` service, you can request your `api key`_ here.

Speedbock Technologies encourages its users to build on our plaforms to improve the user experience and community.

.. _api key: https://www.speedbocktech.com/api

.. note::

   This project is under active development. All information are subject to change.
